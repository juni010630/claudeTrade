# TRI (Trend-Range Index) — 횡보/추세 연속 지수: 채택 (2026-06-11)

요청: "어떤 방법을 써서라도 횡보/추세장 구분지을 수치." 산출물 = **0~100 연속 지수** (BTC+ETH 합성,
1d 주축 + 4h 보조 0.5 가중, 정보용 — 봇 배선 없음, 스위칭은 2026-06-10 전 격자 기각).

## 설계

- **성분 11종** (`regime/trend_index.py`): 신규 — 분산비율 VR(5/10), Hurst, 자기상관합(lag1~5),
  방향일관성(20d), 드리프트 t-stat / 기존 재사용 — Wilder ADX·Choppiness·Kaufman ER·롤링 R²·BB폭 백분위
- **정규화**: 고정 앵커 시그모이드 (VR=1.0, H=0.5, CI=50, ER=0.275 등 정설/통계 기준점 — 데이터 튜닝 없음)
- **합성**: IS 단독 AUC≥0.60 성분만 **등가중 평균** (가중치 학습 없음) → EMA(3d) → 0~100
- **채택 성분 6**: er(.744) tstat(.743) r2(.737) bbwpct(.714) dircon(.663) chop(.643)
  — 탈락: vr5/vr10(.55) acsum(.52) adx_w(.59) hurst(.46 — 역방향, 일봉 크립토에선 무용)
- 라벨: ≥65 추세 / ≤35 횡보 / 사이 = 직전 유지(히스테리시스)

## 방법론 수정 이력 (투명성)

1. **정답 창 ±7d→±15d**: 1차 런에서 ±7d는 2주 잔물결이라 전 성분 AUC ~0.5 희석 → 월 단위 국면으로
   주창 선언, ±7d는 비열위 게이트로 병기 (창 스윕 안 함 — 이 2개 고정)
2. **트레일링 분위수 정규화 → 고정 앵커 시그모이드**: 분위수는 연도 평균을 구조적으로 50에 고정
   (자기 과거 순위)해 수준 비교 불가 → 교체
3. **게이트 ③ "2023 최저" → 월별 Spearman**: 정답 데이터가 전제를 반박 — BTC 2023 실현ER 0.258로
   **최고 추세년**(Q1·Q4 랠리). "2023 횡보"는 봇 6심볼 성과 기준이지 BTC/ETH 가격 기준 아님.
   연 평균은 정답조차 진폭 0.03(노이즈) → 월 단위가 국면의 본래 스케일
